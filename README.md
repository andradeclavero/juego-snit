# Juego SNIT 🗺️🚲🛵

Un juego web interactivo y educativo diseñado para enseñar la importancia y utilidad de los Sistemas de Información Geográfica (SIG / GIS) y los visores territoriales en la vida cotidiana de la ciudad. 

Construido con React, Vite y Tailwind CSS.

## 🎮 Sobre el Juego

El jugador puede elegir entre dos personajes para explorar situaciones urbanas cotidianas donde la información geoespacial es la clave para tomar la mejor decisión:

*   **Bruno (El Ciclista):** Enfocado en movilidad activa, topografía urbana, infraestructura ciclista y seguridad vial.
*   **Olivia (La Exploradora Urbana):** Enfocada en la cultura, ubicación de servicios locales (como ferias libres) y datos de la ciudad (catastro de edificaciones).

Cada personaje tiene 3 niveles o desafíos. El jugador debe elegir la opción correcta basada en el uso inteligente de datos territoriales y mapas colaborativos para avanzar.

## 🛠️ Tecnologías Utilizadas

*   [React 18](https://reactjs.org/) - Librería principal para la interfaz de usuario.
*   [Vite](https://vitejs.dev/) - Entorno de desarrollo rápido y empaquetador web.
*   [Tailwind CSS](https://tailwindcss.com/) - Framework de CSS para diseño rápido y responsivo.
*   JavaScript (ES6+)

## 🚀 Cómo ejecutar el proyecto localmente

Sigue estos pasos para correr el juego en tu computadora:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/andradeclavero/juego-snit.git
   cd juego-snit
   ```

2. **Instalar las dependencias:**
   Necesitas tener [Node.js](https://nodejs.org/) instalado.
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Abrir en el navegador:**
   La terminal te mostrará un enlace (generalmente `http://localhost:5173`). Ábrelo en tu navegador para jugar.

## 📦 Construir para Producción

Para generar los archivos estáticos optimizados para subir a un servidor (producción):

```bash
npm run build
```
Los archivos minificados quedarán en la carpeta `dist`.
